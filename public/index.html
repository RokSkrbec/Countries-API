Endpoints

Below are all available endpoints you can access.

All Countries Endpoint

Returns all the countries in alphabetical order in json format.

/api/countries

Fetch example:

            
              fetch('https://www.countries-api.rokskrbec.si/api/countries')
                .then((res) => res.json())
                .then((data) => {
                  console.log(data);
                })

English Endpoints

Returns single country in json format.

Case insensitive.

Query by short or full country name in English language.

/api/countries/en/ByShortName/:country

Fetch example (Slovenia):

            
              fetch('https://www.countries-api.rokskrbec.si/api/countries/en/ByShortName/slovenia')
                .then((res) => res.json())
                .then((data) => {
                  console.log(data)
                })

result:

            
                {
                "id": "207",
                "enShortName": "Slovenia",
                "enFullName": "Republic of Slovenia",
                "siShortName": "Slovenija",
                "siFullName": "Republika Slovenija",
                "countryCode": "SI"
                }

/api/countries/en/ByFullName/:country

Fetch example (Republic of Slovenia):

            
              fetch('https://www.countries-api.rokskrbec.si/api/countries/en/ByFullName/Republic%20of%20Slovenia')
                .then((res) => res.json())
                .then((data) => {
                  console.log(data)
                })

result:

            
                {
                "id": "207",
                "enShortName": "Slovenia",
                "enFullName": "Republic of Slovenia",
                "siShortName": "Slovenija",
                "siFullName": "Republika Slovenija",
                "countryCode": "SI"
                }

Flag Endpoints

Returns country flag.

The last number indicates flag width.

Widths supported: 64px, 256px, 512px, 1024px, 1920px, 2560px

Image format: PNG

/api/countries/FlagByCountryCode/:countryCode/64
/api/countries/FlagByCountryCode/:countryCode/256
/api/countries/FlagByCountryCode/:countryCode/512
/api/countries/FlagByCountryCode/:countryCode/1024
/api/countries/FlagByCountryCode/:countryCode/1920
/api/countries/FlagByCountryCode/:countryCode/2560
